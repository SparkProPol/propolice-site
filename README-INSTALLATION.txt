PRO POLICE — Pack Decap CMS prêt à installer

1) Dézippez ce pack.
2) Remplacez le contenu du dépôt GitHub par ces fichiers.
3) Le site reste publié par GitHub Pages.
4) L'administration sera à /admin

IMPORTANT :
- La structure CMS est prête.
- Pour pouvoir se connecter EN LIGNE à Decap CMS sur GitHub Pages, il faudra ajouter ensuite un proxy OAuth GitHub.
- En attendant, vous pouvez déjà gérer proprement les contenus dans content/ et uploads/.

Contenus :
- content/data/site.json
- content/data/resources.json
- content/data/articles.json
- content/pages/
- content/articles/
